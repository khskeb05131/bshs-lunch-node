부산고등학교 meal library

인자(파라미터) 소개

/ [time/nut][1/2/3][ /tom]
조,중,석식 구분: time[1/2/3]
영양소, 식사메뉴 구분: [time/nut]
오늘, 내일 구분 [null/tom]

예시: 오늘 점심: /time2 , 오늘 점심식사 영양소: /nut2 , 내일 조식 메뉴: /time1tom

node write.js : ./data 에 급식메뉴 작성
node express.js : ./data 참고하여 host:3000로 표시 (html코드 모두 배제됨)

k-lunch npm library를 사용하여 만들었습니다.
